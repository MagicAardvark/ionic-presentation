Stencil.js

Stencil is not a framework, but a compiler.
Taking in TSX files and producing standard web components

"Why is this so good?"
  • Vanilla JavaScript... no dependencies!
  • Efficient one-way data binding
  • An asynchronous rendering pipeline (similar to React Fiber)
  • Works with any framework. or no framework at all

Ionic V4 will utilize Stencil.js

Ionic V4 will be compatible with Angular, React, VueJS... pretty much what you want!
